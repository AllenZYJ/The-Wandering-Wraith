# Shutter Asylum

A 13kB html5 game challenge.

Made for [Js13kGames](https://js13kgames.com/)

## Getting started
 * `npm install`

## For development
 * `npm run start`

A dev server is started at `http://localhost:8080`

## For production
 * `npm run build`

Ready to use bundle is located in `/dist` directory.
